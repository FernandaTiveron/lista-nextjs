# Lista de Tarefas - Next.js

## Descrição
Aplicação simples de lista de tarefas desenvolvida com Next.js.  
Permite adicionar tarefas, validar entradas e visualizar o total de tarefas cadastradas.

---

## Tecnologias utilizadas
- Next.js
- React
- TypeScript
- Jest
- Testing Library

---

## Como rodar o projeto

Clone o repositório e instale as dependências:

```bash
npm install
```

Execute o projeto em ambiente de desenvolvimento:

```bash
npm run dev
```

Acesse no navegador:  
http://localhost:3000

---

## Como rodar os testes

```bash
npm test
```

---

## Funcionalidades
- Adicionar nova tarefa
- Validação para impedir tarefas vazias
- Limpeza automática do campo após adicionar
- Contador de tarefas
